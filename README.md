# THP Master Portfolio Hub

## Overview
This is the central hub for all my projects created during the **Bootstrap, Atomic Design & Web Marketing** module at **The Hacking Project**.

## Projects Included
1. **The Surfing Project**: Introduction to Bootstrap grids and components.
2. **Community Showcase**: A collaborative directory for peer projects.
3. **Art & Digital Transition**: Strategic landing page applying Atomic Design and Marketing.
4. **Excellence Orchidées**: Expert-level niche landing page (Botany).
5. **Canis Encyclopedia**: Expert-level educational site (Dogs).
6. **Demoscene Archive**: High-tech community portal for digital art.

## Features
- **Centralized Access**: One page to rule them all.
- **Professional Design**: Built with the **Bootswatch Lux** theme for a minimalist, high-end look.
- **AOS Animations**: Smooth entry animations for each project card.
- **Responsive Hub**: Perfect display on all devices.
